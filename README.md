Overview

This project is a web-based virtual lab that simulates fundamental quantum communication protocols. It is designed to make concepts like quantum key distribution, superposition, and entanglement easier to understand through interactive visualizations.

Problem

Quantum communication is difficult to grasp using traditional methods because it is highly theoretical and lacks intuitive learning tools. Students often struggle to visualize how protocols like BB84 and E91 actually work.

Solution

This project provides an interactive platform where users can explore quantum protocols step by step. It combines simulations, visual feedback, and simple UI interactions to bridge the gap between theory and practical understanding.

Features
Simulation of BB84 and E91 protocols
Visualization of quantum states and measurements
Interactive experiments for superposition and entanglement
Basic learning modules with MCQs
Real-time graphs using Plotly
Tech Stack
HTML, CSS, JavaScript
Python, NumPy
Plotly for visualization
How to Run

Clone the repository and open the project locally:

git clone https://github.com/Somyasmith/virtual-qkd-lab-v2.git
cd virtual-qkd-lab-v2

Then open index.html in a browser or use a live server.
